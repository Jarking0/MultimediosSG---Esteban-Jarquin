# Referencias y Fuentes Consultadas

Durante el desarrollo e investigación autónoma para la creación de este proyecto, se consultaron las siguientes fuentes y recursos:

## Documentación Oficial y Tutoriales
1. **Vue.js Official Documentation:** * Introducción a Vue 3: https://vuejs.org/guide/introduction.html
   * Reactividad básica (`ref`, `computed`): https://vuejs.org/guide/essentials/reactivity-fundamentals.html
   * Ciclo de vida (`onMounted`): https://vuejs.org/guide/essentials/lifecycle.html
2. **MDN Web Docs:**
   * Uso de la API Fetch para consumir archivos JSON locales: https://developer.mozilla.org/es/docs/Web/API/Fetch_API
   * Uso de la etiqueta `<audio>` en HTML5: https://developer.mozilla.org/es/docs/Web/HTML/Element/audio

## Recursos Multimedia
Todos los recursos utilizados tienen propósitos estrictamente educativos:
* **Imágenes de los Astros:** NASA Image and Video Library, dominio público (https://images.nasa.gov/).
* **Pistas de Audio:** Descargadas de bancos de audio libres de derechos / Generadas mediante herramientas de Text-to-Speech con fines educativos.

##  Declaración sobre el uso de Inteligencia Artificial
Tal como lo permite el reglamento del "Proyecto Personal v3" del curso IF7102, se hizo uso de herramientas de Inteligencia Artificial (LLMs) estrictamente como apoyo educativo y tutoría. Específicamente, se utilizó para:
* Acelerar la curva de aprendizaje de Vue 3 (Composition API vs Options API).
* Comprender la sintaxis correcta para pasar datos del componente padre (`App.vue`) a los componentes hijos mediante *props* y eventos (`$emit`).
* Estructurar de manera correcta las propiedades computadas (`computed properties`) para lograr el buscador reactivo en tiempo real sin mutar el array original de datos.
Todo el código generado fue analizado, comprendido e integrado manualmente, asegurando mi capacidad para defender y explicar cada línea del proyecto.
